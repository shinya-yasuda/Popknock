# result_recorder
# サービス概要
ポップンミュージックのプレーシェア画像を投稿することでスコアの推移をオプション毎に記録できるwebサービスです。
# 登場人物
* エンドユーザー
  * ポップンミュージックのユーザー
# ユーザーが抱える課題
公式サイトのプレーデータ閲覧機能では各曲のベストスコアは見られるが、スコアの推移は見られないため、以前と比べてどれくらい上達したか分かりにくい。また、譜面を変化させるオプションを使用したか区別されないため、オプションの得手不得手が分かりにくい。
# 解決方法
オブション別でスコアの推移をグラフ化することで上達や得手不得手を可視化する。
# プロダクト
ポップンミュージックのスコア推移を記録できるwebアプリケーション
# マーケット
ポップンミュージックをプレイしているユーザー
